vscode常用按鍵

Posted on 2021-05-08 In 隨意寫寫

快捷鍵

按法 功能 說明
Ctrl+P 開啟檔案 會先開啟一個指令輸入框，輸入工作區的檔案名稱即可開啟檔案
Ctrl+N 新增檔案
Ctrl+W 關閉檔案
Ctrl+Tab 切換檔案視窗 跟在chrom的切換新視窗的效果類似

更快的選擇 - File Utiles

安裝File Utiles擴充功能，開啟指令欄後(Ctrl+Shift+P)，直接輸入想做的事情即可

關鍵字 功能 說明
new file 新增檔案 新增檔案時可先設定新增檔案之目標路徑
delete 刪除檔案 刪除當前開啟的檔案
rename 重新命名 修改當前開啟的檔案的名稱
