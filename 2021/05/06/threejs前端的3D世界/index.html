ThreeJS前端的3D世界

發表於 2021-05-06 更新於 2023-03-30 分類於 網頁開發

環境配置

  • 安裝node.js (開源模組下載)

ThreeJS

Three.js 是一個跨瀏覽器的指令碼，使用 JavaScript 函式庫或 API 來在網頁瀏覽器中建立和展示動畫的 3D 電腦圖形。
git clone https://github.com/mrdoob/three.js.git

  • 開源專案位置
  • 閱讀文件能更完全了解其中設定

Basic step

完成四個步驟 Canvas、Scene、Objects、Materials 後，可進行 Mesh 以呈現出預期的 3D 圖形。
Lights、Sizes、Camera、Renderer 作為補充設定，可調整出不同光線，角度，尺寸下的觀察情境。
Animate 做成動畫讓 3D 的感覺顯示出來。
