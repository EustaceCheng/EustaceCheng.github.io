react免費教程

發表於 2021-06-08 更新於 2023-03-25 分類於 教學

React 是什麼

用於構建用戶介面的 JavaScript
做一個頁面有三個步驟 1.發送請求 2.處理數據 3.呈現頁面(這就是 react 主要在做的事情)

React 特點 1.模組化，聲明是編碼
2.React Native 可用於手機開發 3.虛擬 DOM+Diff 算法，減少直接使用 dom 的操作(效率提高)

React 學前須知 1.判斷 this 的指向
2.class
3.ES6 與法規範
4.npm 5.原型、原型鏈 6.數據常用方法 7.模塊化

jsx 語法規則 1.定義 VDOM，不要寫引號 2.標籤中混入 JS 表達式時要用{} 3.樣式的類名指定不要用 class，要用 className 4.內聯樣式用
{{key:value}} 的形式寫 5.只能有一個根標籤 6.標籤必須閉合 7.標籤首字母若小寫字母開頭，則轉為 html 中同名元素，若無對應同名元素則抱錯；若大寫字母開頭，則渲染對應組件，若組件無定義則抱錯

vscode常用按鍵

發表於 2021-05-08 分類於 隨意寫寫

快捷鍵

按法 功能 說明
Ctrl+P 開啟檔案 會先開啟一個指令輸入框，輸入工作區的檔案名稱即可開啟檔案
Ctrl+N 新增檔案
Ctrl+W 關閉檔案
Ctrl+Tab 切換檔案視窗 跟在chrom的切換新視窗的效果類似

更快的選擇 - File Utiles

安裝File Utiles擴充功能，開啟指令欄後(Ctrl+Shift+P)，直接輸入想做的事情即可

關鍵字 功能 說明
new file 新增檔案 新增檔案時可先設定新增檔案之目標路徑
delete 刪除檔案 刪除當前開啟的檔案
rename 重新命名 修改當前開啟的檔案的名稱

用hexo建立專屬自己的blog

發表於 2021-05-03 更新於 2021-05-19 分類於 教學

環境配置

  • 安裝node.js (開源模組下載)
  • 安裝git (版控、終端機、下載專案等)

Hexo

Hexo是一個快速、簡單且強大的網誌框架。

$ npm install -g hexo-cli

安裝後須先初始化資料夾，在資料夾中加入hexo的初始檔案。

$ hexo init <folder>
$ cd <folder>
$ npm install
  • 在hexo樣板庫挑選樣式，hexo的預設樣式為landscape，本篇用NexT做為範例。

NexT

NexT是Hexo眾多樣板中一個常用樣板

  • 閱讀完參考資料2可開啟更多選單列相關功能
  • 參考官方文件可更了解所有客製化之需求
