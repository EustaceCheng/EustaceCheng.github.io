threejs前端的3D世界

Posted on 2021-05-06

環境配置

  • 安裝node.js (開源模組下載)

threejs

Three.js是一個跨瀏覽器的指令碼，使用JavaScript函式庫或API來在網頁瀏覽器中建立和展示動畫的3D電腦圖形。
git clone https://github.com/mrdoob/three.js.git

  • 開源專案位置
  • 閱讀文件能更完全了解其中設定

basic step

完成四個步驟Canvas、Scene、Objects、Materials後，可進行Mesh以呈現出預期的3D圖形。
Lights、Sizes、Camera、Renderer作為補充設定，可調整出不同光線，角度，尺寸下的觀察情境。
Animate做成動畫讓3D的感覺顯示出來。
