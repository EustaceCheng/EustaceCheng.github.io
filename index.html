React 是什麼

發表於 2021-06-08 更新於 2023-03-30 分類於 前端開發

React 是什麼

用於構建用戶介面的 JavaScript

  • 做一個頁面有三個步驟
    1.發送請求
    2.處理數據
    3.呈現頁面(這就是 react 主要在做的事情)

  • React 特點
    1.模組化，聲明是編碼
    2.React Native 可用於手機開發
    3.虛擬 DOM+Diff 算法，減少直接使用 dom 的操作(效率提高)

  • React 學前須知
    1.判斷 this 的指向
    2.class
    3.ES6 與法規範
    4.npm
    5.原型、原型鏈
    6.數據常用方法
    7.模塊化

  • jsx 語法規則
    1.定義 Virtual DOM，不要寫引號
    2.標籤中混入 JS 表達式時要用{}
    3.樣式的類名指定不要用 class，要用 className
    4.內聯樣式用 {{key:value}} 的形式寫
    5.只能有一個根標籤
    6.標籤必須閉合
    7.標籤首字母若小寫字母開頭，則轉為 html 中同名元素，若無對應同名元素則抱錯；若大寫字母開頭，則渲染對應組件，若組件無定義則抱錯

vscode常用按鍵

發表於 2021-05-08 更新於 2023-03-30 分類於 程式設計

快捷鍵

按法 功能 說明
Ctrl+P 開啟檔案 會先開啟一個指令輸入框，輸入工作區的檔案名稱即可開啟檔案
Ctrl+N 新增檔案
Ctrl+W 關閉檔案
Ctrl+Tab 切換檔案視窗 跟在 Chrome 的切換新視窗的效果類似

更快的選擇 - File Utiles

安裝 File Utiles 擴充功能，開啟指令欄後(Ctrl+Shift+P)，直接輸入想做的事情即可

關鍵字 功能 說明
new file 新增檔案 新增檔案時可先設定新增檔案之目標路徑
delete 刪除檔案 刪除當前開啟的檔案
rename 重新命名 修改當前開啟的檔案的名稱

ThreeJS前端的3D世界

發表於 2021-05-06 更新於 2023-03-30 分類於 網頁開發

環境配置

  • 安裝node.js (開源模組下載)

ThreeJS

Three.js 是一個跨瀏覽器的指令碼，使用 JavaScript 函式庫或 API 來在網頁瀏覽器中建立和展示動畫的 3D 電腦圖形。
git clone https://github.com/mrdoob/three.js.git

  • 開源專案位置
  • 閱讀文件能更完全了解其中設定

Basic step

完成四個步驟 Canvas、Scene、Objects、Materials 後，可進行 Mesh 以呈現出預期的 3D 圖形。
Lights、Sizes、Camera、Renderer 作為補充設定，可調整出不同光線，角度，尺寸下的觀察情境。
Animate 做成動畫讓 3D 的感覺顯示出來。

Hexo

Hexo 是一個快速、簡單且強大的網誌框架。

NexT

NexT 是 Hexo 眾多樣板中一個常用樣板
